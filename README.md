# Mt.Ararat Outdoor Adventures

Bu proje, Mt.Ararat Outdoor Adventures için Next.js ve Vercel CMS kullanılarak oluşturulmuş bir web sitesidir. Ağrı Dağı ve çevresinde sunulan outdoor aktivitelerine dair bilgiler içerir.

## Özellikler

- Next.js 14 App Router
- Typescript
- Vercel CMS entegrasyonu
- Tailwind CSS
- Responsive tasarım
- Admin paneli için kolay içerik yönetimi

## Başlangıç

Projeyi yerel makinenizde çalıştırmak için:

```bash
npm install
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açarak siteyi görüntüleyebilirsiniz.

## CMS Kullanımı

İçerik yönetimi için şu adımları izleyin:

1. Geliştirme sunucusunu başlatın: `npm run dev`
2. Tarayıcıda [http://localhost:3000/cms](http://localhost:3000/cms) adresine gidin
3. CMS panelinden içerikleri düzenleyin

## Dizin Yapısı

- `/cms` - CMS içerik dosyaları
- `/public` - Statik dosyalar (görseller, vb.)
- `/src` - Kaynak kodlar
  - `/app` - Next.js app router sayfaları 
  - `/components` - React bileşenleri
